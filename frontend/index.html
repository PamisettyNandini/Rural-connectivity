Solution: Fog IoT Hub that Works Without Internet

Creates a secure local network for villages — phones connect via Wi‑Fi/BLE to send/receive messages, SOS, and verified alerts.

🧩 Offline Mesh

Device‑to‑device routing forms a village‑scale network.

⚡ Low‑Power & Portable

Battery/solar‑friendly; deploy in minutes.

🔔 Broadcast Alerts

Panchayat/relief officers can push priority messages.

🛡️ Secure by Default

Local auth, signed alerts, role‑based access.

📶 Multi‑Radio

Wi‑Fi, BLE, LoRa* options for wider coverage.

📊 Edge AI*

On‑device triage: classify requests & prioritize SOS.

*Optional modules based on deployment needs.

How It Works (3 Steps)

1

Deploy Hub

Place hub at a school/center. Power via battery/solar. It auto‑boots the local network.

2

Connect Phones

Citizens join Wi‑Fi/BLE SSID. App/PWA opens offline. No mobile data required.

3

Coordinate

Send SOS, share updates, and receive official alerts. Data syncs to cloud when back online.
